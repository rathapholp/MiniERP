# MiniERP - Claude Instructions

## Project
Mini ERP system. GitHub: https://github.com/rathapholp/MiniERP

## Git Workflow
- Always push to GitHub after creating or modifying files
- Remote: `origin` → `https://github.com/rathapholp/MiniERP.git`
- Default branch: `master`
- Git user: `rathapholp` / `rathapholp@users.noreply.github.com`

## Commit convention
- Keep commit messages short and descriptive
- Stage specific files, not `git add .` (avoid accidentally committing secrets)
